  • Backend
    • 3 types of database relationships
      • One to One: One record in one table links to another record in another table. The one is usually called a primary key.
      • One to Many: One record in one table links to multiple records in another table. The many is usually called a foreign key.
      • Many to Many: Multiple records in one table links to multiple records in another table. This requires a junction table.
